## 주요 신규 기능 & 실전 활용

### v2.1.72 — effort 레벨 간소화 & 개발자 편의성 대폭 개선 (3/10)

v2.1.72에서 effort 레벨이 기존 4단계(low/medium/high/max)에서 **3단계(low/medium/high)**로 간소화되었습니다. 상태바 표시도 ○(low), ◐(medium), ●(high)로 변경되어 현재 설정을 직관적으로 확인할 수 있습니다.

**주요 변경사항:**

- **`/plan` 즉시 진입**: `/plan fix the auth bug`처럼 설명 인자를 직접 전달하면 즉시 플랜 모드에 진입합니다
- **`/copy`에 `w` 키 추가**: 선택 항목을 클립보드 대신 파일로 직접 저장 — SSH 원격 환경에서 특히 유용합니다
- **`ExitWorktree` 도구 추가**: worktree 세션을 도구로 종료할 수 있습니다
- **bash 자동 승인 확대**: `lsof`, `pgrep`, `tput`, `ss`, `fd`, `fdfind` 추가
- **번들 사이즈 ~510KB 축소**: 전반적인 로딩 속도가 개선됩니다
- **음성 전사 개선**: 레포지토리 이름과 개발 용어 인식 정확도가 향상되었습니다

```bash
# 즉시 플랜 모드 진입
/plan fix the auth bug

# SSH에서 파일로 직접 저장
/copy  # → w 키를 눌러 파일에 쓰기
```
