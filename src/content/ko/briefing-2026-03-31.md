### 플리커 프리 렌더링 — `CLAUDE_CODE_NO_FLICKER=1` (v2.1.88)

긴 세션에서 화면이 깜빡이는 문제를 해결하는 새로운 렌더링 모드가 추가되었습니다. 가상화된 스크롤백과 alt-screen 렌더링을 사용하여 플리커를 완전히 제거합니다.

```bash
# 환경변수로 활성화
export CLAUDE_CODE_NO_FLICKER=1
```

[GitHub v2.1.88](https://github.com/anthropics/claude-code/releases)

### PermissionDenied 훅 (v2.1.88)

Auto 모드에서 분류기가 명령을 거부했을 때 발생하는 새로운 훅 이벤트입니다. `{retry: true}`를 반환하면 재시도를 허용할 수 있습니다. 거부된 명령은 알림으로 표시되고 `/permissions` Recent 탭에서 확인 가능합니다.

```json
{
  "hooks": {
    "PermissionDenied": [
      {
        "command": "echo '거부된 명령 로깅' >> /tmp/denied.log"
      }
    ]
  }
}
```

### 명명 서브에이전트 @ 멘션 지원 (v2.1.88)

이름이 지정된 서브에이전트가 `@` 멘션 자동완성 목록에 표시됩니다. 팀 에이전트를 이름으로 직접 참조하여 메시지를 보낼 수 있어 멀티에이전트 워크플로우가 더 직관적으로 개선되었습니다.

### Claude Code의 15가지 숨겨진 강력 기능

Claude Code 창시자 Boris Cherny가 잘 알려지지 않은 15가지 기능을 정리하여 화제가 되고 있습니다.

- **`--teleport`**: 세션을 다른 머신으로 이동
- **`/loop` & `/schedule`**: 반복 작업 자동화와 cron 스케줄링
- **`/branch`**: 세션을 분기하여 대안 탐색
- **`/btw`**: 메인 작업 중단 없이 사이드 질문
- **`/batch`**: 여러 에이전트를 병렬 실행
- **`--bare`**: 10배 빠른 스크립트 모드 시작
- **`--add-dir`**: 멀티 리포지토리 동시 작업
- **`/voice`**: 음성으로 코딩 지시
- **Chrome 확장**: 브라우저에서 직접 Claude Code 세션 관리

## 개발자 워크플로우 팁

### Claude Code로 보안 코드 리뷰 수행하기

보안 전문 기업 SpecterOps의 Andrew Luke가 Claude Code를 애플리케이션 보안 평가의 '힘의 배율기(force multiplier)'로 활용하는 방법을 공개했습니다.

**핵심 기법:**
- 공격자 관점의 보안 중심 시스템 프롬프트 설계
- 데몬 시스템 매핑, Cypher 쿼리 처리 추적, 보안 컨트롤 식별에 활용
- 프라이빗 코드는 로컬 배포 또는 Amazon Bedrock 사용 권장 — 공개 Claude.ai 사용 비권장

```markdown
# 보안 리뷰용 CLAUDE.md 예시
- 모든 사용자 입력 경로를 추적하고 sanitization 여부를 확인할 것
- SQL 쿼리, 시스템 명령, 파일 경로에서 인젝션 가능성을 우선 검토
- 인증/인가 바이패스 경로를 체계적으로 탐색
```

[SpecterOps Blog](https://specterops.io/blog/2026/03/26/leveling-up-secure-code-reviews-with-claude-code/)

### 컨텍스트 관리가 생산성의 핵심

실전 프로젝트 기반 베스트 프랙티스 가이드에서 컨텍스트를 가장 중요한 자원으로 강조합니다.

- **세션 분리**: 작업 단위로 새 세션 시작, 무관한 작업 간 `/clear` 사용
- **70% 규칙**: 컨텍스트 사용량이 70%에 도달하면 선제적으로 `/compact` 실행
- **리서치 위임**: 탐색적 작업은 서브에이전트에 위임하여 메인 컨텍스트 보호
- **훅 > CLAUDE.md**: 매번 반드시 실행되어야 하는 작업(포매팅, 린팅, 보안 검사)은 CLAUDE.md가 아닌 훅으로 설정 — 훅은 결정적(100%), CLAUDE.md는 권고 사항

[Ran The Builder](https://ranthebuilder.cloud/blog/claude-code-best-practices-lessons-from-real-projects/) | [MorphLLM](https://www.morphllm.com/claude-code-best-practices)
