## 주요 신규 기능 & 실전 활용

### Code Review for Claude Code — 멀티 에이전트 PR 리뷰 시스템 (3/9 출시)

Anthropic이 Claude Code에 **Code Review** 기능을 리서치 프리뷰로 출시했습니다. PR이 열리면 여러 에이전트가 병렬로 코드를 분석하여 로직 오류, 보안 취약점, 엣지 케이스, 미묘한 회귀를 탐지합니다.

**핵심 특징:**
- **멀티 에이전트 구조**: 여러 에이전트가 병렬로 버그를 탐색하고, 오탐을 필터링한 후 심각도별로 순위를 매깁니다
- **리뷰 결과**: PR에 단일 요약 코멘트 + 개별 인라인 코멘트로 전달
- **실효성 입증**: 내부 테스트에서 실질적 리뷰 코멘트가 달리는 PR 비율이 16%에서 54%로 증가
- **평균 리뷰 시간**: 약 20분
- **가격**: PR당 평균 $15–25

현재 Claude Team 및 Claude Enterprise 고객 대상 리서치 프리뷰로 제공됩니다. AI 생성 코드가 급증하는 환경에서 코드 품질을 유지하기 위한 실질적인 도구입니다.

[Anthropic 공식](https://claude.com/blog/code-review) | [TechCrunch](https://techcrunch.com/2026/03/09/anthropic-launches-code-review-tool-to-check-flood-of-ai-generated-code/)

### Claude Partner Network — $1억 투자와 파트너 생태계 구축 (3/12)

Anthropic이 **Claude Partner Network**을 출범시키며 2026년 초기 투자로 **$1억(약 1,300억원)**을 투입합니다. 파트너사에 교육, 기술 지원, 공동 마케팅을 제공하여 엔터프라이즈 Claude 도입을 가속화합니다.

**주요 내용:**
- 가입비 무료, Claude를 시장에 가져오는 모든 조직이 대상
- 파트너 대응팀을 **5배 확충**: 전담 Applied AI 엔지니어, 기술 아키텍트, 해외 시장 지원
- **Claude Certified Architect, Foundations**: 프로덕션 애플리케이션 구축을 위한 최초의 기술 자격증 출시
- **Code Modernization 스타터 키트**: 레거시 코드베이스 마이그레이션과 기술 부채 해소를 위한 파트너용 도구

Claude Code를 활용한 컨설팅이나 기업 도입 프로젝트를 고려하는 파트너사에게 직접적인 기회입니다.

[Anthropic 공식](https://www.anthropic.com/news/claude-partner-network) | [VentureBeat](https://venturebeat.com/technology/anthropic-rolls-out-code-review-for-claude-code-as-it-sues-over-pentagon)

### Claude for Excel & PowerPoint — 앱 간 공유 컨텍스트와 원클릭 스킬 (3/11)

Claude for Excel과 Claude for PowerPoint에 **앱 간 공유 컨텍스트**와 **재사용 가능한 스킬** 기능이 추가되었습니다.

**핵심 기능:**
- **공유 컨텍스트**: Excel과 PowerPoint 파일을 하나의 대화에서 동시에 참조하며 작업 — 한쪽 앱의 작업이 다른 쪽에도 반영
- **원클릭 스킬**: 반복되는 워크플로우를 저장하여 조직 전체에서 재사용 가능
- **프리로드 스타터 스킬**: Excel에서는 수식 오류 감사·DCF/LBO 템플릿·데이터 정리, PowerPoint에서는 경쟁 분석 덱·IB 자료 검토 등

Mac과 Windows에서 유료 Claude 플랜 사용자에게 베타로 제공되며, Amazon Bedrock, Google Cloud Vertex AI, Microsoft Foundry를 통한 LLM 게이트웨이 접근도 지원합니다.

[Anthropic 공식](https://claude.com/blog/claude-excel-powerpoint-updates) | [VentureBeat](https://venturebeat.com/orchestration/anthropic-gives-claude-shared-context-across-microsoft-excel-and-powerpoint)

개발자 커뮤니티에서 **LLM 멀티 에이전트 워크플로우**를 활용한 오픈소스 제로데이 취약점 발견 사례가 공유되었습니다(GeekNews 16포인트). Mozilla Firefox 사례에 이어, AI 기반 보안 분석이 실전에서 재현 가능한 방법론으로 자리잡고 있음을 보여줍니다.

Claude Code의 `/review` 명령이나 새로운 Code Review 기능과 결합하면, 자체 프로젝트의 보안 검토 워크플로우를 체계화할 수 있습니다.

[GeekNews](https://news.hada.io/)

### Python 공급망 보안 — 심층 방어(Defense in Depth) 가이드

파이썬 패키지 생태계의 공급망 공격에 대응하기 위한 **다계층 방어 전략** 가이드가 공유되었습니다. pip install로 시작하는 대부분의 프로젝트에서 의존성 보안은 과소평가되기 쉬운 영역입니다. Claude Code로 `requirements.txt`나 `pyproject.toml`의 의존성을 분석하고 알려진 취약점을 점검하는 것도 효과적인 첫 단계입니다.

## 흥미로운 프로젝트 & 도구

- **AutoKernel** (14포인트): PyTorch 모델을 입력받아 **Triton/CUDA 커널을 자동으로 최적화**하는 GPU 커널 자동 연구 시스템입니다. ML 모델의 추론 성능을 수작업 없이 개선할 수 있으며, 커스텀 모델 서빙 환경에서 특히 유용합니다. [GeekNews](https://news.hada.io/)

- **BitNet** (1포인트): Microsoft의 **1비트 LLM 추론 프레임워크** bitnet.cpp입니다. CPU와 GPU 모두에서 빠르고 손실 없는 추론을 지원하며, 엣지 환경에서의 LLM 배포 비용을 극적으로 낮출 수 있습니다. [GeekNews](https://news.hada.io/)

- **ralph-loop** (13포인트): Claude를 활용한 **장시간 실행형 AI 에이전트 루프** 시스템입니다. 반복 작업 목록을 Docker 샌드박스 환경에서 자동으로 순회하며, Claude Code의 자동화 워크플로우를 확장하는 데 참고할 수 있습니다. [GeekNews](https://news.hada.io/)
