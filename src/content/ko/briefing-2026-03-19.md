## 주요 신규 기능 & 실전 활용

### Claude Code v2.1.79 — 원격 제어와 안정성 개선 (3/18)

v2.1.78에 이어 같은 날 릴리스된 **v2.1.79**는 개발 경험(DX) 개선과 안정성에 초점을 맞췄습니다.

**주요 변경:**
- **`--console` 인증 플래그** — `claude auth login --console`으로 Anthropic Console(API 과금) 인증 직접 수행 가능. 팀/엔터프라이즈 환경에서 인증 워크플로우 간소화
- **턴 소요시간 표시** — `/config` 메뉴에 "Show turn duration" 토글 추가. 각 턴이 얼마나 걸렸는지 실시간 확인
- **시작 시 메모리 사용량 ~18MB 절감** — 모든 시나리오에서 메모리 효율 향상
- **비스트리밍 API 폴백 개선** — 시도당 2분 타임아웃 추가로 세션이 무한 대기하는 문제 방지
- **`CLAUDE_CODE_PLUGIN_SEED_DIR` 다중 경로 지원** — Unix에서 `:`, Windows에서 `;`로 여러 시드 디렉토리 지정 가능
- **`claude -p` 서브프로세스 행 수정** — Python `subprocess.run` 등에서 명시적 stdin 없이 호출 시 행이 발생하던 문제 해결
- **Ctrl+C `-p` 모드 수정** — print 모드에서 Ctrl+C가 정상 동작
- **`/btw` 스트리밍 중 응답 수정** — 스트리밍 중 `/btw`가 메인 에이전트 출력 대신 사이드 질문에 답변
- **`SessionEnd` 훅 수정** — 인터랙티브 `/resume`으로 세션 전환 시 `SessionEnd` 훅이 정상 발동
- **보이스 모드 시작 수정** — `voiceEnabled: true` 설정 시 시작 시 보이스 모드가 정상 활성화

**VSCode 전용:**
- **`/remote-control`** — 세션을 claude.ai/code로 브릿지하여 브라우저나 폰에서 이어서 작업 가능
- **AI 생성 세션 탭 제목** — 첫 메시지를 기반으로 세션 탭 제목 자동 생성
- **Thinking pill 수정** — 응답 완료 후 "Thinking" 대신 "Thought for Ns" 표시
- **사이드바 세션 diff 버튼 수정**

[GitHub Releases](https://github.com/anthropics/claude-code/releases)

## 개발자 워크플로우 팁

### v2.1.79 `/remote-control` — 브라우저에서 세션 이어가기

VSCode에서 작업 중인 Claude Code 세션을 claude.ai/code로 브릿지하여, 자리를 비울 때 폰이나 다른 기기에서 이어서 작업할 수 있습니다.

```
# VSCode Claude Code 패널에서
/remote-control
```

### `--console` 플래그로 API 인증 간소화

팀 환경에서 Anthropic Console(API 과금) 인증을 CLI에서 직접 수행:

```bash
claude auth login --console
```

### 턴 소요시간 확인하기

`/config` → "Show turn duration" 활성화 → 각 턴의 소요시간을 실시간으로 확인하여 프롬프트 최적화에 활용.
