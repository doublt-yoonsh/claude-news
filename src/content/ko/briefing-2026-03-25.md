## 주요 신규 기능 & 실전 활용

### Anthropic vs. Pentagon — Judge Rita Lin, "Anthropic을 무력화하려는 시도"로 보인다

어제(3/24) 샌프란시스코 연방법원에서 열린 가처분 심리에서 Rita Lin 판사가 정부 측에 **날카로운 회의적 질문**을 쏟아냈습니다.

**Judge Lin의 핵심 발언:**
- "살인인지는 모르겠지만, **Anthropic을 무력화하려는 시도**로 보인다"
- "이 세 가지 조치가 명시된 국가안보 우려에 맞춤화되어 있다고 보기 어렵다"
- 정부의 '사보타주' 이론에 대해: "그 연결고리가 보이지 않는다"

**정부 측 주요 시인 사항:**
- Hegseth 국방장관의 의회 서한에 **덜 제한적인 대안 검토가 누락**됨을 인정
- Anthropic이 실제로 군 시스템에 무단 업데이트를 푸시할 수 있는지 **DOD도 모른다**고 시인 (감사 진행 중)
- Anthropic 엔지니어의 "배포 후 모델 중단 능력 없음" 선서에 대한 반박 없음

**향후 일정:**
- Judge Lin은 **수일 내 결정** 예고 — Anthropic은 3/26까지 판결 요청
- 정부 측은 금지명령 발부 시 **항소 계류 중 정지(stay)** 신청 예고

## 개발자 워크플로우 팁

### Claude Code Channels — Telegram/Discord에서 Claude Code 조작

Claude Code Channels가 리서치 프리뷰로 출시되었습니다. Telegram이나 Discord에서 직접 Claude Code 세션에 메시지를 보낼 수 있습니다.

**작동 방식:**
- 플러그인 기반 — 로컬 Claude Code 세션에 Telegram/Discord 메시지 전달
- 세션은 파일시스템, MCP, git 등 전체 접근 권한으로 요청 처리 후 같은 앱으로 응답
- Slack, WhatsApp, iMessage 지원은 커뮤니티에서 요청 중

```bash
# Telegram에서 Claude Code에 메시지 보내기
# "src/api/users.ts에서 인증 로직 리팩토링해줘"
# → 로컬 Claude Code가 실행하고 결과를 Telegram으로 회신
```

[VentureBeat](https://venturebeat.com/orchestration/anthropic-just-shipped-an-openclaw-killer-called-claude-code-channels) | [Claude Code Docs](https://code.claude.com/docs/en/channels)

### 커뮤니티 베스트 프랙티스 — "프롬프트가 아니라 에이전트로 생각하라"

Hacker News와 r/ClaudeCode에서 떠오르는 패턴:

- **Composable Skills**: 집중된 `SKILL.md` 파일 + 서브에이전트 조합이 새로운 베스트 프랙티스
- **명확한 스펙 우선**: 한 개발자가 12단계 구현 문서 작성에 2시간 투자 → Claude가 단계별로 정확히 실행
- **r/ClaudeCode**: 주간 기여자 4,200명 이상, 경쟁 도구 대비 3배 이상

## 알아두면 좋은 소소한 변경사항

- **가처분 심리 핵심 쟁점 정리**: 정부는 Anthropic의 이용약관 거부가 수정헌법 1조로 보호되지 않는 "상업적 행위"라고 주장했으나, 판사는 회의적 반응을 보였습니다.
- **Computer Use 안전 주의**: Anthropic은 여전히 민감 정보(비밀번호, 금융 데이터) 처리에 Computer Use를 사용하지 말 것을 권장합니다.
- **Channels vs OpenClaw**: Anthropic의 Channels는 공식 지원과 안전성에서, OpenClaw는 커뮤니티 확장성에서 각각 강점을 보입니다.
- **3/23 claude.ai 경미한 인시던트**: 16분 만에 해결된 에러율 상승 — 공식 장애 공지 없이 조용히 해결됨.
