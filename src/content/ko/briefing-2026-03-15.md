## 개발자 워크플로우 팁

### v2.1.76 MCP Elicitation — 실전 활용 가이드

어제 출시된 MCP Elicitation은 MCP 서버가 작업 중 사용자에게 구조화된 입력을 요청하는 기능입니다. 활용 시나리오:

- **배포 파이프라인**: MCP 서버가 배포 대상 환경, 버전, 롤백 조건 등을 폼으로 확인
- **데이터베이스 마이그레이션**: 스키마 변경 전 영향 범위를 사용자에게 확인 요청
- **OAuth 인증 흐름**: 브라우저 URL을 통한 인증 다이얼로그 지원

`Elicitation`/`ElicitationResult` 훅으로 응답을 가로채거나 자동화할 수 있어, CI/CD 환경에서의 비대화형 워크플로우에도 적용 가능합니다.

### 중국 AI 기업의 대규모 Distillation 공격 — 24,000개 위장 계정 적발

Anthropic이 **DeepSeek, MiniMax, Moonshot AI** 3개 중국 기업의 조직적 distillation 공격을 공개했습니다.

**규모:**
- 총 **24,000개 위장 계정**, **1,600만 건 이상의 Claude 쿼리**

- 상업 프록시 서비스를 통해 중국 접속 제한을 우회
- 일반 고객 트래픽에 distillation 요청을 섞어 탐지를 회피

Claude의 에이전틱 코딩, 추론, 도구 사용 능력이 주요 타깃이었습니다. API 키 관리와 이상 트래픽 모니터링의 중요성을 다시 한번 확인시키는 사건입니다.
