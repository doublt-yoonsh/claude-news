## 주요 신규 기능 & 실전 활용

### Cowork Dispatch 메시지 전달 버그 해결 (v2.1.87 + Desktop 1.1.9493)

Claude Desktop 1.1.9310 버전에서 Dispatch 세션이 응답하지 않는 심각한 버그가 발견되어 수정되었습니다.

**문제 상황:**
- 모바일/데스크톱에서 Dispatch로 메시지를 보내면 수신·처리는 되지만 응답이 전달되지 않음
- 데스크톱 백엔드는 정상 작동하나 Cloud Control Relay(CCR)와 클라이언트 간 릴레이 계층에서 문제 발생
- GitHub에 다수의 버그 리포트(#40178, #40179, #40257, #40283) 접수

**해결:**
- Claude Code v2.1.87에서 메시지 전달 로직 수정
- Claude Desktop 1.1.9493으로 업데이트 필요

```bash
# Claude Code 업데이트
brew upgrade --cask claude-code

# Claude Desktop 업데이트 확인
# 설정 > 업데이트에서 1.1.9493 이상 확인
```

[GitHub v2.1.87](https://github.com/anthropics/claude-code/releases) | [GitHub Issue #40283](https://github.com/anthropics/claude-code/issues/40283)

## 개발자 워크플로우 팁

### Claude Code 필수 스킬 3선 — 시작이 반이다

Claude Code 스킬 생태계가 급성장하여 150개 이상의 커뮤니티 스킬과 72개 이상의 공식 플러그인이 등록되어 있습니다. 처음 시작한다면 이 3가지부터 설치하는 것이 권장됩니다.

**1. frontend-design (277,000+ 설치)**
Anthropic 공식 스킬. 코드를 작성하기 전에 디자인 시스템과 철학을 먼저 수립합니다. 대담한 미학, 의도적인 타이포그래피, 목적 있는 색상 팔레트를 생성합니다.

**2. simplify**
작성된 코드의 재사용성, 품질, 효율성을 리뷰하고 문제를 수정합니다. 코드 리뷰 루프를 자동화하는 가장 간단한 방법입니다.

**3. browser-use**
Claude를 헤드리스 브라우저에 연결합니다. URL 탐색, 폼 작성, 동적 콘텐츠 스크래핑, 스크린샷, 배포 기능 E2E 검증까지 가능합니다.
