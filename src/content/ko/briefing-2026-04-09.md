## 주요 신규 기능 & 실전 활용

### Focus View 토글 — Ctrl+O (v2.1.97)

v2.1.97에서 NO_FLICKER 모드에 **Focus View**가 추가되었습니다. `Ctrl+O`로 토글하면 프롬프트, 도구 호출 한 줄 요약(편집 diff 통계 포함), 최종 응답만 표시됩니다. 복잡한 멀티스텝 작업에서 중간 과정의 시각적 노이즈를 줄이고 결과에 집중할 수 있습니다.

```bash
# NO_FLICKER 모드 활성화 (필수 전제)
export CLAUDE_CODE_NO_FLICKER=1

# 실행 후 Ctrl+O로 Focus View 토글
claude
```

[GitHub v2.1.97](https://github.com/anthropics/claude-code/releases/tag/v2.1.97)

### Claude Managed Agents API 출시 (4/9)

Anthropic이 대규모 에이전트를 프로그래매틱하게 배포할 수 있는 **Managed Agents** API 플랫폼을 출시했습니다. 에이전트 구성, 배포, 모니터링을 단일 API로 처리할 수 있는 구성 가능한(composable) 플랫폼이며, Notion, Rakuten, Asana, Sentry가 초기 도입 사례로 참여합니다. 세션당 $0.08/시간으로 가격이 책정되었습니다.

## 개발자 워크플로우 팁

### agent-skills — Addy Osmani의 프로덕션급 AI 코딩 스킬 모음

Google Cloud AI 디렉터 Addy Osmani가 19개 프로덕션급 엔지니어링 스킬을 오픈소스로 공개했습니다. 7개 슬래시 명령으로 구성되며, Hyrum's Law("관찰 가능한 모든 행동에 누군가는 의존한다"), Chesterton's Fence("이유를 이해하기 전에 제거하지 마라") 같은 Google 엔지니어링 원칙이 AI 에이전트 워크플로우에 내장되어 있습니다.

- **구성**: 7개 슬래시 명령, 19개 스킬
- **특징**: 각 스킬에 프로덕션 엔지니어링 원칙이 포함되어, AI가 "기술적으로 동작하지만 프로덕션에서 문제가 되는" 코드를 생성하는 것을 방지
- **설치**: `git clone https://github.com/addyosmani/agent-skills`

```bash
# Claude Code에서 슬래시 명령으로 사용
/review    # 코드 리뷰
/plan      # 구현 계획
/test      # 테스트 전략
```

### Awesome Design.MD — AI에게 디자인 시스템을 전달하는 표준 포맷

60+ 서비스(Claude, Vercel, Stripe, Notion, Spotify)의 디자인 시스템을 마크다운으로 정리한 컬렉션입니다. 프로젝트 루트에 `DESIGN.md`를 두면 AI 에이전트가 해당 서비스의 디자인 언어를 참조하여 일관된 UI를 생성합니다. 9개 표준화된 섹션(색상, 타이포그래피, 간격, 컴포넌트 등)으로 구성됩니다.

- **CLAUDE.md와의 관계**: CLAUDE.md가 코딩 규칙을 AI에게 전달한다면, DESIGN.md는 디자인 규칙을 전달하는 역할
- **활용**: 원하는 서비스의 디자인 시스템을 복사하여 AI가 해당 스타일로 UI를 생성하도록 유도

## 알아두면 좋은 소소한 변경사항

- **`refreshInterval` 상태 표시줄 설정**: 상태 표시줄 명령을 N초마다 자동 재실행하는 설정이 추가되었습니다
- **`/agents`에 `N running` 표시**: 실행 중인 서브에이전트 수가 에이전트 타입 옆에 표시됩니다
- **Cedar 정책 파일 구문 강조**: `.cedar`, `.cedarpolicy` 파일에 구문 강조가 적용됩니다
- **MCP HTTP/SSE 메모리 누수 수정**: 서버 재연결 시 ~50MB/hr 미해제 버퍼가 누적되던 문제가 수정되었습니다
- **429 재시도 지수 백오프**: 기존에 ~13초에 모든 재시도를 소진하던 문제가 수정되어, 지수 백오프가 최소값으로 적용됩니다
- **서브에이전트 worktree 디렉토리 누수 수정**: worktree 격리 서브에이전트의 작업 디렉토리가 부모 세션의 Bash 도구로 누수되던 문제가 수정되었습니다
- **Context-low 경고 방식 변경**: 영구 행 대신 일시적 푸터 알림으로 표시됩니다
- **Bash 도구 OTEL 트레이싱**: 트레이싱 활성화 시 하위 프로세스가 W3C `TRACEPARENT` 환경변수를 상속합니다
