### Anthropic vs. Pentagon — Judge Lin, 예비적 금지명령 발부 (3/26)

Judge Rita Lin이 트럼프 행정부의 Anthropic 사용 금지 조치를 차단하는 **예비적 금지명령(preliminary injunction)**을 발부했습니다. 지난 주간 매일 추적해온 이 소송의 첫 번째 실질적 결과입니다.

**판결 핵심:**
- 정부의 금지 조치가 Anthropic의 **수정헌법 제1조로 보호되는 표현**에 대한 처벌인지 의문을 제기
- 명령은 정부 항소를 위해 **7일간 정지(stay)**
- 정부는 즉각 항소 예고

AI 기업의 군사 계약 거부권에 대한 산업 전체 선례가 될 전망입니다.

[Bloomberg](https://www.bloomberg.com/news/articles/2026-03-26/anthropic-wins-court-order-blocking-trump-ban-on-use-of-ai-tool) | [CBS News](https://www.cbsnews.com/news/anthropic-ruling-judge-trump-pentagon-ai/) | [Axios](https://www.axios.com/2026/03/26/judge-temporarily-blocks-pentagon-ban-anthropic)

### Auto Mode — Claude가 스스로 권한을 판단하는 새 모드 (3/24)

Anthropic이 **Auto Mode**를 리서치 프리뷰로 출시했습니다. 분류기(classifier)가 각 도구 호출의 안전성을 실시간 평가하여 파일 쓰기, Bash 명령 등을 자동 승인합니다.

**작동 방식:**
- 안전한 작업(파일 읽기, 검색): 자동 승인
- 위험한 작업(대량 삭제, 민감 데이터 접근, 악성 코드 실행): 사용자 확인 요청
- 기존 `--permission-mode auto`와 달리, 분류기가 각 호출을 개별 평가

**대상:** Team 플랜 (Enterprise, API 추후 지원)
**지원 모델:** Sonnet 4.6, Opus 4.6

[Anthropic Blog](https://claude.com/blog/auto-mode) | [TechCrunch](https://techcrunch.com/2026/03/24/anthropic-hands-claude-code-more-control-but-keeps-it-on-a-leash/)

### v2.1.84 — Windows PowerShell 도구 (프리뷰)

Windows 사용자를 위한 **네이티브 PowerShell 도구**가 opt-in 프리뷰로 추가되었습니다. 기존 Bash 도구와 병행하여 Windows 네이티브 명령 실행이 가능합니다.

[PowerShell 도구 문서](https://code.claude.com/docs/en/tools-reference#powershell-tool)

### v2.1.85 — Hook 조건부 `if` 필터

훅에 `if` 필드가 추가되어 권한 규칙 문법(`Bash(git *)` 등)으로 실행 조건을 지정할 수 있습니다. 모든 도구 호출마다 훅 프로세스를 생성하는 오버헤드를 줄여줍니다.

```json
{
  "hooks": {
    "PreToolUse": [
      {
        "if": "Bash(git *)",
        "command": "echo 'git 명령어 감지'"
      }
    ]
  }
}
```
