### Anthropic vs. Pentagon — Judge Lin, 예비적 금지명령 인용 (3/26)

**어제 가장 큰 뉴스입니다.** Judge Rita Lin이 43페이지 판결문으로 Anthropic의 예비적 금지명령을 **인용**했습니다.

**판결 핵심:**
- 국방부의 Anthropic 공급망 위험 지정 **차단**
- 트럼프 대통령의 전 연방기관 Anthropic 기술 사용 중단 명령 **차단**
- 단, 국방부에 Anthropic 제품 사용을 **강제하지는 않음** — 타 AI 제공업체로의 합법적 전환은 허용

**판결문 핵심 문구:**
- "정부와의 의견 불일치를 표현했다는 이유로 미국 기업을 잠재적 적국이자 방해자로 낙인찍는 것을 뒷받침하는 법적 근거는 없다"
- "Anthropic이 정부의 계약 입장에 대해 공적 관심을 불러일으킨 것을 처벌하는 것은 전형적인 수정헌법 1조 보복"

**향후 전망:**
- Lin 판사는 정부의 항소를 위해 **7일간 집행 유예** 부여 (4월 2일경까지)
- 정부 측은 유예 기간 내 항소 예상 — 최종 판결까지 수개월 소요 가능
- Anthropic: "법원이 신속하게 행동해주신 것에 감사하며, 본안에서 승소할 가능성이 높다는 데 동의해주셨다"

## 개발자 워크플로우 팁

### Claude Code v2.1.85 — 조건부 훅과 성능 개선

어제 하루 만에 두 버전(v2.1.84, v2.1.85)이 연달아 출시되었습니다.

**v2.1.85 주요 변경 (3/26):**
- **조건부 훅**: 훅에 `if` 필드 추가 — 권한 규칙 문법(`Bash(git *)`)으로 특정 도구 호출 시에만 훅 실행, 프로세스 스폰 오버헤드 감소
- **MCP 멀티서버**: `CLAUDE_CODE_MCP_SERVER_NAME`, `CLAUDE_CODE_MCP_SERVER_URL` 환경변수로 `headersHelper` 스크립트가 서버 구분 가능
- **트랜스크립트 타임스탬프**: `/loop` 또는 `CronCreate` 예약 작업 실행 시 타임스탬프 표시
- **스크롤 성능**: WASM yoga-layout을 순수 TypeScript로 교체 — 대형 트랜스크립트 렌더링 개선
- **MCP OAuth**: RFC 9728 Protected Resource Metadata 디스커버리 지원
- **버그 수정**: `/compact` "context exceeded" 에러, non-git 저장소에서 `--worktree`, 원격 세션 메모리 누수, 지속적 ECONNRESET 에러, 멀티 모니터 `switch_display` 등

```jsonc
// v2.1.85 조건부 훅 예시 — git 명령어에만 훅 실행
{
  "hooks": {
    "PreToolUse": [
      {
        "if": "Bash(git *)",
        "command": "echo 'Git command detected'"
      }
    ]
  }
}
```

**v2.1.84 주요 변경 (3/26):**
- **Windows PowerShell 도구** (옵트인 프리뷰): 최초의 네이티브 Windows 셸 도구
- **모델 capability 오버라이드**: `ANTHROPIC_DEFAULT_{OPUS,SONNET,HAIKU}_MODEL_SUPPORTS` 환경변수 — Bedrock/Vertex/Foundry에서 고정 모델 사용 시 기능 플래그 수동 설정
- **유휴 복귀 프롬프트**: 75분 이상 자리 비운 후 `/clear` 안내
- **`x-client-request-id` 헤더**: API 요청에 디버깅용 요청 ID 추가
- **VSCode**: 레이트 리밋 경고 배너에 사용률(%)과 리셋 시간 표시

[v2.1.85 Release](https://github.com/anthropics/claude-code/releases/tag/v2.1.85) | [v2.1.84 Release](https://github.com/anthropics/claude-code/releases/tag/v2.1.84)
