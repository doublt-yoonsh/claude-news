## 주요 신규 기능 & 실전 활용

### Anthropic, Trump 행정부 상대로 두 건의 연방 소송 제기 — "공급망 리스크" 지정 철회 요구 (3/9)

Anthropic이 3월 9일(월) Pentagon의 "공급망 리스크(supply chain risk)" 지정을 철회하기 위해 **두 건의 연방 소송**을 제기했습니다. 하나는 캘리포니아 북부 연방지방법원에, 다른 하나는 워싱턴 D.C. 연방항소법원에 접수되었습니다.

**핵심 쟁점:**
- Pentagon은 Anthropic이 자사 기술의 **무제한 군사 사용**을 허용하지 않자 공급망 리스크로 지정
- Anthropic은 **대규모 국내 감시**와 **완전 자율 치명 무기** 두 가지 레드라인을 고수
- Pentagon은 "국가안보 비상시 민간 기업이 사용 방식을 제한할 수 없다"는 입장
- 이 지정은 통상 **외국 적대 세력** 관련 기업에 적용되는 것으로, Anthropic의 국방부 관련 사업 전반에 영향

**Anthropic의 법적 주장:**
48페이지 분량의 소장에서 "이러한 조치는 전례 없고 위법"이라며, 정부가 **보호받는 표현(speech)을 이유로 기업을 처벌하는 것은 헌법에 위반**된다고 주장했습니다. 재무부, 국무부 등 다른 연방 기관도 피고로 포함되었으며, 이들은 직원들에게 Anthropic 서비스 사용 중단을 지시한 것으로 알려졌습니다.

### Claude Marketplace 출시 — 엔터프라이즈 AI 도구 조달의 새 판 (3/6)

Anthropic이 **Claude Marketplace**를 한정 프리뷰로 출시했습니다. 기업 고객이 기존 API 지출 약정의 일부를 Claude 기반 서드파티 애플리케이션 구매에 사용할 수 있는 B2B 마켓플레이스입니다.

**핵심 특징:**
- **수수료 0%** — Anthropic은 서드파티 거래에서 수수료를 받지 않음
- AWS Marketplace, Azure Marketplace와 유사한 모델이지만, AI 네이티브 생태계에 특화
- 수십 개 벤더 계약 대신 **단일 조달 채널**로 통합
- 런칭 파트너: **GitLab, Harvey, Lovable, Replit, Rogo, Snowflake**

SaaS 조달 프로세스의 간소화를 목표로 하지만, **벤더 종속(vendor lock-in)** 우려도 제기되고 있습니다. Anthropic은 "고객이 이미 약정한 지출을 더 유연하게 사용할 수 있게 하는 것"이라고 설명합니다.

## 개발자 워크플로우 팁

### CLAUDE.md는 이제 .gitignore만큼 필수다

2026년 기준 Claude Code 전문가들이 공통으로 강조하는 베스트 프랙티스를 정리했습니다.

**1. 계획 먼저, 코드는 나중에:**
- Plan 모드에서 계획을 반복 수정한 뒤, 승인 후에만 auto-accept로 전환
- "Claude가 코드를 쓰기 전에 작성된 계획을 검토하고 승인하는 것이 **가장 중요한 한 가지**"

**2. 컨텍스트 위생 관리:**
- 작업당 새 세션 시작, 관련 없는 작업 사이에 `/clear` 사용
- 컨텍스트 사용량 70%에서 선제적으로 compact
- 리서치는 서브에이전트에 위임하여 메인 컨텍스트 보호

**3. 검증 기준 제공:**
- "Claude에게 검증 기준을 주는 것이 **가장 높은 레버리지의 실천**" — 테스트 실행, 스크린샷 비교, 출력 검증을 스스로 수행할 수 있을 때 성능이 극적으로 향상

**4. 커스텀 명령어 활용:**
- `.claude/commands/`의 커스텀 명령어가 **가장 큰 생산성 향상** — `/new-feature`, `/deploy`, `/test-all` 등 멀티스텝 워크플로우를 단일 명령으로

잘 구성된 프로젝트(CLAUDE.md + 커스텀 명령어 + 적절한 프롬프팅)는 기본 설정 대비 **5~10배 빠른** 기능 출시가 가능합니다.

## 보안/제한 이슈

### 중국 AI 연구소 3곳, Claude에서 "산업 규모" 증류 공격 — 2.4만 위조 계정, 1,600만 교환 (2/23)

Anthropic이 **DeepSeek, Moonshot AI, MiniMax** 세 중국 AI 기업의 대규모 모델 증류(distillation) 캠페인 증거를 공개했습니다. 이 사건은 Pentagon 분쟁의 배경으로도 중요합니다.

**공격 규모:**
- **24,000개 위조 계정**, 상업용 프록시 서비스를 통해 접근
- **1,600만 회 이상**의 Claude 상호작용 생성
- IP 주소 상관관계, 요청 메타데이터, 인프라 지표로 각 연구소를 특정

**연구소별 타겟:**
- **DeepSeek**: 추론 능력, 채점 태스크, 정치적 민감 질문의 검열 우회 대안 생성 (15만+ 교환)
- **Moonshot AI**: 에이전틱 추론, 도구 사용, 코딩, 컴퓨터 비전 (340만+ 교환)
- **MiniMax**: 에이전틱 코딩 및 도구 사용 (1,300만+ 교환)

Anthropic은 "불법 증류된 모델은 안전 가드레일이 누락되어 사이버 범죄, 생물무기, 대규모 감시에 악용될 수 있다"고 경고했습니다.

### Claude, App Store 1위 등극 — Pentagon 분쟁이 만든 역설적 성장

Claude 앱이 3월 초 **Apple App Store와 Google Play에서 동시 1위**를 달성했습니다. 2월 초 App Store 42위에서 수일 만에 정상에 오르는 급격한 상승세를 보였습니다.

**상승 경로:** 42위(2월 초) → 6위(수요일) → 4위(목요일) → **1위(토요일, 3/1)**

이 현상의 배경에는 Reddit과 X에서 퍼진 **"Cancel ChatGPT"** 운동이 있습니다. OpenAI의 Pentagon 계약에 반발한 사용자들이 ChatGPT 계정 삭제 가이드를 공유하고 Claude로 이동을 독려했습니다. Bloomberg가 보도한 **런레이트 매출 190억 달러 돌파**와 맞물려, Pentagon 분쟁이 Anthropic에게는 역설적으로 **가장 효과적인 브랜드 빌딩 이벤트**가 되었습니다.

## 알아두면 좋은 소소한 변경사항

- **v2.1.71이 여전히 최신 버전**: 3/10 기준 신규 릴리스가 없습니다. `/loop` 명령, cron 스케줄링, VS Code 세션 관리 등이 포함된 v2.1.71이 최신입니다. `brew upgrade --cask claude-code`로 업데이트하세요.
- **VS Code 향상 사항 (v2.1.69~71)**: Activity bar에 spark 아이콘으로 Claude Code 세션 목록 확인, 계획의 전체 마크다운 문서 뷰, 네이티브 MCP 서버 관리 다이얼로그가 추가되었습니다.
- **API 안정성 개선**: `ANTHROPIC_BASE_URL`로 서드파티 게이트웨이 사용 시 발생하던 API 400 오류가 수정되었으며, 여러 Claude Code 인스턴스를 동시에 실행할 때 플러그인 설치가 사라지는 문제도 해결되었습니다.
