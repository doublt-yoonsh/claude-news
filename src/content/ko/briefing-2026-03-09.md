## 주요 신규 기능 & 실전 활용

### Anthropic, AI 노동시장 영향 보고서 발표 — "위협받는 것은 저임금 직종이 아니라 고소득 전문직" (3/5)

Anthropic 연구진(Maxim Massenkoff, Peter McCrory)이 AI의 노동시장 영향을 측정하는 새로운 프레임워크와 초기 증거를 발표했습니다. Claude의 실제 업무 사용 데이터를 기반으로 한 **"관측 노출도(observed exposure)"**라는 새로운 측정 방법을 도입하여, 이론적 가능성과 실제 도입 사이의 격차를 정량화했습니다.

**핵심 발견:**
- **컴퓨터 프로그래머가 가장 높은 노출도**(75% 태스크 커버리지)를 보임 — 이어서 고객 서비스 담당자, 데이터 입력 전문가
- 고노출 직종 근로자는 여성 비율이 16% 더 높고, 평균 임금이 47% 더 높으며, 대학원 학위 보유율이 4배 높음
- 전체 근로자의 **30%는 AI 노출도가 0**
- **이론 vs 현실의 격차**: 컴퓨터/수학 직종은 이론적 AI 커버리지가 94%이지만, 실제 관측 노출도는 33%에 불과

Fortune은 이 보고서를 인용하여 **"화이트칼라 대불황(Great Recession)"이 현실적 가능성**이라고 보도했습니다. 다만 현재까지 대규모 실업 증가는 관측되지 않았으며, 22~25세 연령대에서 고노출 직종의 채용이 약 14% 감소하는 신호가 포착되었습니다. **해고보다 채용 축소**를 통한 점진적 전환이 진행 중입니다.

## 개발자 워크플로우 팁

### 파일은 인간과 에이전트가 상호작용하는 인터페이스다

파일 시스템이 AI 에이전트 시대에 핵심 인터페이스로 재부상하고 있다는 분석 글이 화제입니다. Claude Code의 `CLAUDE.md`, `SKILL.md` 같은 파일 기반 포맷이 API나 중앙 집중형 조율 없이 플랫폼 간 기능을 공유할 수 있는 **개방형 표준**으로 진화하고 있다는 주장입니다.

**Claude Code 사용자를 위한 실전 인사이트:**
- ETH Zurich 연구에 따르면, **과도한 컨텍스트 파일은 오히려 성능을 저하**시킴 — 최소한의 핵심 정보만 담는 것이 효과적
- 제약 조건이 저장 용량에서 **컨텍스트 관리 품질**로 이동 — 파일의 양보다 질이 중요
- 파일 기반 접근은 SaaS 생태계 대비 데이터 이동성과 사용자 소유권을 보장

**실전 적용:** `CLAUDE.md`를 작성할 때 프로젝트의 모든 것을 넣기보다, **가장 중요한 컨벤션과 아키텍처 결정 사항만 간결하게** 유지하는 것이 Claude Code의 성능을 높입니다. 100줄짜리 CLAUDE.md가 500줄짜리보다 더 나은 결과를 만들 수 있습니다.

### LLM Writing Tropes — AI 글쓰기의 클리셰를 시스템 프롬프트로 차단하기

AI가 생성하는 텍스트의 반복적인 언어 패턴을 **6가지 카테고리**로 체계적으로 정리한 참조 문서가 공개되었습니다. 시스템 프롬프트에 추가하면 기계적인 글쓰기 스타일을 억제할 수 있습니다.

**주요 패턴 예시:**
- **단어**: "quietly", "deeply", "robust", "tapestry", "ecosystem" 등 강조어·기업 용어 남발
- **구조**: "It's not X — it's Y" 대조법, 극적인 카운트다운, 과도한 반복
- **어조**: "Here's the thing"으로 시작하는 거짓 긴장감, 인위적 취약성

핵심 원칙: **"인간처럼 쓰라: 다양하고, 불완전하고, 구체적으로."** Claude Code로 README, 문서, 블로그를 생성할 때 이 tropes 목록을 `CLAUDE.md`의 글쓰기 규칙에 포함하면 더 자연스러운 출력을 얻을 수 있습니다.

## 생태계 & 플러그인

### Claude Code 한국어 플레이북 — 59챕터, 5단계 레벨별 무료 가이드

GeekNews Show에서 Claude Code의 포괄적인 한국어 학습 가이드가 공개되었습니다. 공식 문서가 영어 중심인 문제를 해결하기 위해 제작되었으며, 설치부터 엔터프라이즈 팀 구축까지 59개 챕터로 구성되어 있습니다.

**5단계 레벨 구성:**
- **Level 1 (입문)**: 설치, API 키, IDE 설정
- **Level 2 (기초)**: CLAUDE.md, 슬래시 명령, Git 워크플로우
- **Level 3 (중급)**: Hooks, MCP 서버, Skills, 비용 최적화
- **Level 4 (고급)**: 커스텀 명령, Agent SDK, 대규모 코드베이스
- **Level 5 (마스터)**: 멀티 에이전트 오케스트레이션, 엔터프라이즈 팀 구축

Worktree, Agent Teams, Desktop App, Fast Mode 등 최신 기능까지 다루며, 캡스톤 프로젝트("실제 엔터프라이즈 AI 에이전트 팀 구축")와 재사용 가능한 프롬프트·용어집 부록도 포함합니다. 한국어로 Claude Code를 처음 시작하거나 체계적으로 학습하고 싶은 개발자에게 강력히 추천합니다.
