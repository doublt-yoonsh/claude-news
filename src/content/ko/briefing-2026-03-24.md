### Claude Computer Use — Mac에서 Claude가 직접 컴퓨터를 조작 (3/23 출시)

Anthropic이 **Claude Computer Use** 기능을 macOS에 출시했습니다. Claude가 사용자의 Mac을 직접 조작하여 작업을 수행할 수 있습니다.

**주요 특징:**
- **직접 조작**: Claude가 앱 열기, 브라우저 탐색, 스프레드시트 작성, 파일 편집 등을 마우스 클릭과 키보드 입력으로 수행
- **커넥터 우선**: Google Workspace, Slack 등 지원 서비스는 커넥터 우선 사용, 미지원 서비스는 직접 조작
- **권한 요청**: 민감한 작업 전 사용자에게 허가 요청 (민감 정보 취급 시 주의 권장)
- **Dispatch 연동**: 모바일에서 Claude에 지시 → Mac에서 작업 실행하는 원격 워크플로우 가능
- **대상**: Claude Pro 및 Max 구독자 (리서치 프리뷰)

### Computer Use + Claude Code 조합 워크플로우

어제 출시된 Computer Use와 Claude Code를 조합하면 새로운 자동화가 가능합니다.

```bash
# Claude Code로 코드 작성 후 → Computer Use로 브라우저 테스트
# 예: API 구현 → Postman/브라우저에서 자동 검증
# Dispatch로 모바일에서 지시 → Mac에서 전체 워크플로우 실행
```
