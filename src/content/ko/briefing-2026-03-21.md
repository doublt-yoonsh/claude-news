## 주요 신규 기능 & 실전 활용

### Claude Code v2.1.81 — 스크립트 실행 최적화와 모바일 권한 중계 (3/20)

v2.1.80 다음 날 릴리스된 **v2.1.81**은 자동화 시나리오와 원격 권한 관리에 초점을 맞췄습니다.

**주요 변경:**
- **`--bare` 플래그** — 훅, LSP, 플러그인 싱크, 스킬 디렉토리 탐색을 모두 스킵하는 경량 모드. `ANTHROPIC_API_KEY` 또는 `--settings`의 `apiKeyHelper` 필수. OAuth·키체인 인증과 자동 메모리 비활성화. 스크립트/자동화 호출에 최적
- **`--channels` 권한 중계** — 권한 기능을 선언한 채널 서버가 도구 승인 프롬프트를 휴대폰으로 전달. 원격 세션에서 모바일로 권한 승인 가능
- **OAuth 동시 세션 수정** — 여러 Claude Code 세션이 동시에 실행될 때 하나의 세션이 OAuth 토큰을 갱신하면 다른 세션이 재인증을 요구하던 문제 해결
- **보이스 모드 안정화** — 재시도 실패를 무시하고 오해의 소지가 있는 "네트워크를 확인하세요" 메시지를 표시하던 문제 수정. 서버가 WebSocket 연결을 조용히 끊을 때 오디오가 복구되지 않던 문제 해결
- **`CLAUDE_CODE_DISABLE_EXPERIMENTAL_BETAS` 수정** — structured-outputs 베타 헤더를 억제하지 못해 Vertex/Bedrock 프록시에서 400 오류 발생하던 문제 해결
- **Node.js 18 크래시 수정** — Node.js 18 환경에서의 충돌 문제 해결
- **Bash 명령어 불필요 권한 프롬프트 수정** — 문자열에 대시(-)가 포함된 Bash 명령어에 불필요한 권한 요청이 나타나던 문제 해결
- **워크트리 세션 복원** — 워크트리에서 세션을 재개하면 해당 워크트리로 자동 전환
- **MCP 도구 호출 표시 개선** — MCP read/search 도구 호출이 단일 "Queried {server}" 줄로 축소 표시 (Ctrl+O로 확장)
- **`!` bash 모드 발견성 개선** — 인터랙티브 명령어 실행 시 Claude가 `!` 모드를 제안
- **플러그인 최신성 개선** — ref 추적 플러그인이 매 로드 시 재클론하여 업스트림 변경 반영
- **Remote Control 세션 개선** — 제목 자동 생성, `/rename` 동기화, `/exit` 아카이브 안정화
- **MCP OAuth CIMD 지원** — Dynamic Client Registration 없는 서버를 위한 Client ID Metadata Document(SEP-991) 지원
- **플랜 모드 변경** — "clear context" 옵션 기본 숨김 (`"showClearContextOnPlanAccept": true`로 복원)
- **Windows 줄단위 스트리밍 비활성화** — 렌더링 문제로 Windows(WSL 포함)에서 줄단위 응답 스트리밍 비활성화

## 개발자 워크플로우 팁

### v2.1.81 `--bare` — 자동화 스크립트에서 Claude Code 경량 호출

CI/CD 파이프라인이나 자동화 스크립트에서 Claude Code를 호출할 때 `--bare` 플래그로 불필요한 초기화를 스킵할 수 있습니다.

```bash
ANTHROPIC_API_KEY=sk-... claude -p "이 코드를 리뷰해줘" --bare
```

훅, LSP, 플러그인 싱크, 스킬 디렉토리 탐색을 모두 건너뛰어 빠르고 깔끔한 실행이 가능합니다.

### `--channels` 권한 중계로 모바일에서 승인

원격 서버에서 Claude Code를 실행하면서 도구 승인을 휴대폰으로 받을 수 있습니다. 채널 서버가 권한 기능을 선언하면 승인 프롬프트가 모바일로 전달됩니다.

### MCP 도구 호출 축소 표시

v2.1.81에서 MCP read/search 도구 호출이 단일 줄로 축소되어 출력이 깔끔해졌습니다. 상세 내용은 Ctrl+O로 확장 가능.

## 알아두면 좋은 소소한 변경사항

- **v2.1.81 플랜 모드 "clear context" 기본 숨김**: 플랜 승인 후 컨텍스트 클리어 옵션이 기본적으로 숨겨짐. `"showClearContextOnPlanAccept": true`로 복원 가능.
- **v2.1.81 Windows 줄단위 스트리밍 비활성화**: 렌더링 문제로 Windows 및 WSL에서 줄단위 응답 스트리밍 비활성화.
- **v2.1.81 Remote Control 세션 제목**: 제네릭 제목 대신 첫 프롬프트 기반 제목 자동 생성. 세 번째 메시지 후 갱신.
- **v2.1.81 tmux 클립보드 문제 수정**: 빠른 Cmd+Tab 후 붙여넣기 시 클립보드 내용이 손실되던 문제 해결.
- **v2.1.81 `/btw` 수정**: 활성 응답 중 사용 시 붙여넣은 텍스트가 포함되지 않던 문제 해결.
- **v2.1.81 투명 훅 첨부파일**: 보이지 않는 훅 첨부파일이 트랜스크립트 모드에서 메시지 수를 부풀리던 문제 해결.
