## 주요 신규 기능 & 실전 활용

### DOJ, Anthropic의 수정헌법 제1조 주장 정면 반박 — 40페이지 서면 제출 (3/17, 금주 핵심)

법무부(DOJ)가 캘리포니아 연방법원에 **40페이지 분량의 반박 서면**을 제출하며 Anthropic의 가처분 신청 기각을 촉구했습니다. 3/24 심리를 앞두고 양측의 법적 공방이 최고조에 달했습니다.

**DOJ 핵심 주장:**
- **"행위이지 표현이 아니다"**: Anthropic이 정부 계약 조건을 거부한 것은 수정헌법 제1조가 보호하는 '표현'이 아니라 '행위'라고 주장
- **"보복 동기가 있더라도 결과는 동일"**: 설령 보복적 동기가 있었다 하더라도, 국가안보 우려라는 정당한 근거만으로도 동일한 조치를 취했을 것이라고 주장
- **"이용약관이 수용 불가해졌다"**: Anthropic이 자율 무기와 대규모 감시에 대한 제한을 고집한 후 그 이용약관이 행정부에 수용 불가해졌다고 명시

- **"워크플로우는 매우 유사"**: 경쟁 모델 간 워크플로우가 비슷해 전환 혼란이 최소화될 것이라고 주장
- **"단일 제공자 의존이 문제"**: "하나의 주요 제공자에 의존하는 것은 국방부에 적합하지 않다"

- **180일 전환 기한**: 방위 업체들도 DoD 작업에서 Claude를 사용하지 않음을 인증해야 함

## 개발자 워크플로우 팁

### Claude Code Channels — 5분 만에 Telegram으로 제어하기

v2.1.80에서 리서치 프리뷰로 도입된 Channels를 활용하면 Telegram이나 Discord에서 직접 Claude Code 세션에 메시지를 보낼 수 있습니다.

```bash
# 채널 서버와 함께 Claude Code 시작
claude --channels
```

MCP 기반이므로 커뮤니티가 Slack, WhatsApp 등 추가 커넥터를 직접 구축할 수 있습니다. v2.1.81의 `--channels` 권한 중계로 모바일에서 도구 승인까지 가능합니다.

### `--bare` 모드 — CI/CD 자동화의 핵심

```bash
ANTHROPIC_API_KEY=sk-... claude -p "이 PR을 리뷰해줘" --bare
```

훅, LSP, 플러그인 싱크, 스킬 디렉토리 탐색을 모두 건너뛰어 자동화 환경에서 빠르고 깔끔한 실행이 가능합니다.
