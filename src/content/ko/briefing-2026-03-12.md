## 주요 신규 기능 & 실전 활용

### v2.1.73 — modelOverrides, /simplify·/batch 번들 명령, HTTP 훅 (3/11)

v2.1.73에서 `modelOverrides` 설정이 추가되어 모델 선택기 항목을 커스텀 프로바이더 모델 ID(예: Bedrock 추론 프로필 ARN)에 매핑할 수 있습니다. 기업 환경에서 자체 모델 엔드포인트를 사용하는 개발자에게 특히 유용합니다.

**주요 변경사항:**

- **`/simplify` 번들 명령**: 변경된 코드를 자동으로 검토하여 재사용성, 품질, 효율성을 점검하고 개선합니다
- **`/batch` 번들 명령**: 여러 작업을 일괄 처리할 수 있는 배치 명령이 추가되었습니다
- **HTTP 훅 지원**: 셸 명령 대신 URL로 JSON을 POST하고 JSON 응답을 받는 HTTP 기반 훅을 설정할 수 있습니다
- **worktree 간 설정 공유**: 동일 저장소의 git worktree 간에 프로젝트 설정과 auto memory가 공유됩니다
- **VS Code 세션 관리 개선**: 액티비티 바에 스파크 아이콘이 추가되어 모든 Claude Code 세션을 목록으로 확인하고 에디터로 열 수 있습니다
- **플랜 마크다운 뷰**: VS Code에서 플랜을 전체 마크다운 문서로 보고 코멘트로 피드백을 남길 수 있습니다
- **Opus 4.6 기본 effort 변경**: Max/Team 구독자의 Opus 4.6 기본 effort가 medium으로 설정됩니다

```bash
# modelOverrides 예시 (settings.json)
# "modelOverrides": { "opus": "arn:aws:bedrock:..." }

# 코드 품질 자동 개선
/simplify

# 여러 작업 일괄 실행
/batch
```

### Microsoft, Anthropic 소송 지지 — 아미쿠스 브리프 제출 (3/10–11)

Microsoft가 Anthropic의 국방부 소송을 지지하는 **아미쿠스 브리프(법정 조언서)**를 제출했습니다. Microsoft는 "Anthropic 파트너로서 국방부의 지정에 직접적 영향을 받는다"고 밝히며, 법원에 **일시적 금지 명령**을 촉구했습니다.

**핵심 논점:**

- **미군 AI 활용 중단 우려**: 즉각적인 계약 변경이 강제되면 미군의 진행 중인 AI 활용이 **"방해(hampered)"**될 수 있다고 경고
- **업계 연대 확산**: OpenAI·Google DeepMind 직원 약 40명(Google DeepMind 수석 과학자 Jeff Dean 포함)도 별도의 지지 서한 제출
- **법률 전문가 분석**: 미네소타 대학 법학대학 Alan Rozenshtein 교수는 "해당 법률이 외국 연루가 없는 미국 기업에 적용될 수 있는지조차 불분명"하다고 분석

법률 전문가들은 Anthropic이 **강력한 소송 근거**를 갖추고 있다고 평가합니다. 정부가 동시에 (1) 국방생산법으로 서비스 판매를 강제하면서 (2) 실제 군사 작전에서 사용하고 (3) 공급망 리스크로 지정하는 것은 자기모순이라는 지적입니다.

## 개발자 워크플로우 팁

### Claude 3/11 장애 사후 분석 — 개발자가 배워야 할 의존성 관리 교훈

3월 11일 Claude AI가 전 세계적으로 약 2시간 동안 장애를 겪었습니다. Downdetector 기준 **10,000명 이상**이 문제를 보고했으며, Claude Code는 **OAuth 인증 장애**로 개발자들이 작업 중 잠금 상태에 빠졌습니다. 단, Claude API는 정상 작동했습니다.

**개발자를 위한 교훈:**

- **API vs. 웹/CLI 분리 인식**: Claude API는 정상이었지만 Claude.ai와 Claude Code CLI가 영향을 받았습니다. 프로덕션에서는 API 직접 호출이 더 안정적입니다
- **로컬 폴백 전략**: Claude Code가 다운되더라도 작업을 이어갈 수 있도록 git stash, 로컬 노트, 대체 도구 전략을 마련해두세요
- **클라우드 단일 의존성 리스크**: 엔터프라이즈 환경에서는 AI 코딩 도구의 장애에 대비한 워크플로우 설계가 필요합니다

### MCP, 월간 SDK 다운로드 9,700만 돌파 — AI 도구 연결의 사실상 표준

Anthropic이 2024년 11월에 공개한 **Model Context Protocol(MCP)**이 월간 SDK 다운로드 **9,700만 건**을 돌파하며 AI 도구 연결의 사실상 표준으로 자리잡았습니다. Claude, ChatGPT, Cursor, Gemini, Microsoft Copilot, VS Code 등 주요 클라이언트가 모두 MCP를 지원합니다.

2025년 12월 Anthropic이 MCP를 **Linux Foundation 산하 Agentic AI Foundation(AAIF)**에 기증한 이후, Google, Microsoft, AWS, Cloudflare, Bloomberg 등이 공동 지원하고 있습니다. 1만 개 이상의 활성 서버가 운영 중이며, "AI의 USB-C"라는 평가가 나올 정도로 빠르게 확산되고 있습니다.

### Anthropic, 중국 AI 연구소의 대규모 모델 탈취 폭로

Anthropic이 중국 AI 3사 — **DeepSeek, Moonshot AI, MiniMax** — 가 Claude를 **산업적 규모로 증류(distillation)**했다는 증거를 공개했습니다. 총 **24,000개의 위조 계정**으로 **1,600만 건 이상의 상호작용**을 수행했습니다.

**주요 세부사항:**

- **DeepSeek**: 15만 건 이상의 추론 과제, 강화학습 보상 모델용 채점, 정치적 민감 쿼리 검열 우회 재작성에 집중
- **Moonshot**: 340만 건 이상의 에이전틱 추론, 도구 사용, 코딩, 데이터 분석, 컴퓨터 비전 관련 상호작용
- **"하이드라 클러스터"**: 단일 프록시 설정이 2만 개 이상의 위조 계정을 동시에 제어하며 추출 트래픽을 일반 사용과 혼합

Anthropic은 **행동 지문(behavioral fingerprinting)**과 체인 오브 소트 추출 패턴 감지 시스템을 배포하여 방어를 강화했습니다.

## 알아두면 좋은 소소한 변경사항

- **v2.1.73 OAuth SSL 오류 가이드 추가**: 기업 프록시나 `NODE_EXTRA_CA_CERTS` 환경에서 로그인이나 연결 체크가 실패할 때 구체적인 해결 방법을 안내합니다.
- **v2.1.73 `/rename` 개선**: Claude가 처리 중일 때도 `/rename`이 즉시 동작합니다(기존에는 묵시적으로 큐에 쌓였음).
- **v2.1.73 프롬프트 입력 리렌더 ~74% 감소**: 턴 진행 중 프롬프트 입력부의 불필요한 리렌더가 대폭 줄어들었습니다.
- **v2.1.73 시작 메모리 ~426KB 절감**: 커스텀 CA 인증서가 없는 사용자의 시작 메모리가 절감됩니다.
- **v2.1.73 Remote Control 폴링 빈도 축소**: 연결 상태에서 10분당 1회로 줄여 서버 부하를 약 300배 감소시켰습니다.
- **v2.1.73 네이티브 MCP 서버 관리 대화상자**: VS Code에서 MCP 서버를 네이티브 UI로 관리할 수 있습니다.
