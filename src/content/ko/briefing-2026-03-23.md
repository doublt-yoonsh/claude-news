## 개발자 워크플로우 팁

### Claude Code Channels — 실전 자동화 패턴

Channels가 출시 1주일을 맞으며 커뮤니티에서 실전 활용 패턴이 정립되고 있습니다.

**자동 배포 알림 패턴:**
```bash
# CI/CD 파이프라인에서 배포 완료 후 Claude에 알림
claude --channels &
# Telegram/Discord로 "배포 완료, 스모크 테스트 실행해줘" 메시지 전송
```

**모바일 코드 리뷰 패턴:**
```bash
# --channels + --bare 조합으로 원격 리뷰
ANTHROPIC_API_KEY=sk-... claude --channels --bare
# 이동 중 Telegram에서 PR 리뷰 요청 → 결과 수신
```

v2.1.81의 권한 중계로 모바일에서 도구 승인까지 가능하므로, 완전한 원격 워크플로우가 실현됩니다.
