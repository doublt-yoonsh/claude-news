## 主要な新機能と実践活用

### Anthropic vs. Pentagon — Judge Lin、予備的差止命令を認容（3/26）

**昨日最大のニュースです。** Judge Rita Linが43ページの判決文でAnthropicの予備的差止命令を**認容**しました。

**判決の核心：**
- 国防総省のAnthropic供給チェーンリスク指定を**差止**
- トランプ大統領の全連邦機関Anthropic技術使用停止命令を**差止**
- ただし、国防総省にAnthropic製品の使用を**強制するものではない** — 他のAIプロバイダーへの合法的な移行は許容

**判決文の主要引用：**
- 「政府との意見の不一致を表明したことを理由に、米国企業を潜在的な敵対者および破壊工作員として烙印を押すことを支持する法的根拠は、関連法規のどこにもない」
- 「Anthropicが政府の契約姿勢に対して公的な注目を集めたことを罰することは、典型的な違法な修正第1条報復である」

**今後の展望：**
- Lin判事は政府の控訴のために**7日間の執行猶予**を付与（4月2日頃まで）
- 政府側は猶予期間内に控訴予定 — 最終判決まで数ヶ月かかる可能性
- Anthropic：「裁判所が迅速に行動してくださったことに感謝し、本案で勝訴する可能性が高いことに同意していただけた」

## 開発者ワークフローティップス

### Claude Code v2.1.85 — 条件付きフックとパフォーマンス改善

昨日一日で2バージョン（v2.1.84、v2.1.85）が連続リリースされました。

**v2.1.85の主要変更（3/26）：**
- **条件付きフック**: フックに`if`フィールド追加 — パーミッションルール構文（`Bash(git *)`）で特定のツール呼び出し時のみフック実行、プロセススポーンのオーバーヘッド削減
- **MCPマルチサーバー**: `CLAUDE_CODE_MCP_SERVER_NAME`、`CLAUDE_CODE_MCP_SERVER_URL`環境変数で`headersHelper`スクリプトがサーバーを識別可能
- **トランスクリプトタイムスタンプ**: `/loop`または`CronCreate`予約タスク実行時にタイムスタンプを表示
- **スクロール性能**: WASM yoga-layoutをピュアTypeScriptに置き換え — 大規模トランスクリプトのレンダリング改善
- **MCP OAuth**: RFC 9728 Protected Resource Metadataディスカバリーサポート
- **バグ修正**: `/compact`の「context exceeded」エラー、non-gitリポジトリでの`--worktree`、リモートセッションのメモリリーク、持続的ECONNRESETエラー、マルチモニター`switch_display`など

```jsonc
// v2.1.85 条件付きフックの例 — gitコマンドにのみフック実行
{
  "hooks": {
    "PreToolUse": [
      {
        "if": "Bash(git *)",
        "command": "echo 'Git command detected'"
      }
    ]
  }
}
```

**v2.1.84の主要変更（3/26）：**
- **Windows PowerShellツール**（オプトインプレビュー）：初のネイティブWindowsシェルツール
- **モデルcapabilityオーバーライド**: `ANTHROPIC_DEFAULT_{OPUS,SONNET,HAIKU}_MODEL_SUPPORTS`環境変数 — Bedrock/Vertex/Foundryでの固定モデル使用時に機能フラグを手動設定
- **アイドル復帰プロンプト**: 75分以上離席後に`/clear`を案内
- **`x-client-request-id`ヘッダー**: APIリクエストにデバッグ用リクエストID追加
- **VSCode**: レート制限警告バナーに使用率（%）とリセット時間を表示
