## 主要な新機能と実践活用

### Skills Marketplaceエコシステムが本格拡大 — 423プラグイン、2,849スキル（4/26）

Claude Codeのプラグイン・スキルエコシステムが急速に成長しています。コミュニティ主導のマーケットプレイスで**423プラグイン、2,849スキル、177エージェント**が公開されており、Anthropic公式マーケットプレイス以外にも複数のサードパーティハブが運営されています。

核心は**Execution Runtime**パターンです。コードベース全体をコンテキストに読み込む代わりに、ClaudeがローカルでPythonスクリプトを実行してAPIにアクセスする方式で、トークン消費を大幅に削減します。スキルは`SKILL.md`ファイルにYAMLフロントマターと指示を定義すれば、会話コンテキストに応じて自動的にトリガーされます。

```bash
# プラグインの検索とインストール
claude plugin search "code-review"
claude plugin install @marketplace/code-review-pro

# 個人スキルの作成
mkdir -p ~/.claude/skills
# → SKILL.mdにトリガー条件と指示を定義
```

`blockedMarketplaces`と`strictKnownMarketplaces`設定で、組織内で承認されたマーケットプレイスのみに制限できます。「Anthropic Verified」バッジ付きプラグインは手動の品質・安全レビューを通過しています。

[BrightCoding](https://www.blog.brightcoding.dev/2026/04/26/claude-skills-marketplace-the-essential-plugin-hub-for-developers) | [GitHub](https://github.com/jeremylongshore/claude-code-plugins-plus-skills)

## 開発者ワークフローティップス

### Context7プラグインでドキュメントのハルシネーションを排除

Claudeがフレームワークの誤ったAPIを生成する最大の原因は、トレーニングデータのバージョン不一致です。**Context7プラグイン**はライブラリドキュメントをバージョン別にインデックス化し、Claudeが必要なページだけを正確に取得できるようにします。

Next.js 15、React 19、Tailwind CSS 4など、急速に進化するフレームワークを使用する際、Context7が現行バージョンの公式ドキュメントを直接提供するため、deprecatedパターンや存在しないAPIの生成が大幅に減少します。

```json
// settings.jsonにContext7 MCPサーバーを追加
{
  "mcpServers": {
    "context7": {
      "command": "npx",
      "args": ["-y", "@upstash/context7-mcp"]
    }
  }
}
```

Claudeにオンラインドキュメントを直接検索させるより、Context7経由の方が効率的です。fetch・パース・フィルタリングのプロセスが省略され、トークンと時間の両方を節約できます。

[Context7 GitHub](https://github.com/upstash/context7) | [Claude Plugin](https://claude.com/plugins/context7)

### ツールは少ないほど良い —「厳選ツールキット」の原則

MCPサーバーやプラグインが増える中、「ツール過負荷」が現実的な問題になっています。Marmelabチームの実戦経験によると、各ツールはコンテキストに複雑性を追加し、管理負担を増やします。

**実践ルール**: 週3回以上使用するツールだけをアクティブにし、残りは無効化してください。半端に使う10個のツールより、厳選した4〜5個の方がはるかに効果的です。プロジェクトごとに`.claude/settings.local.json`で必要なMCPサーバーだけを有効化することを推奨します。

[Marmelab](https://marmelab.com/blog/2026/04/24/claude-code-tips-i-wish-id-had-from-day-one.html)

米国政府がClaude Mythosの**連邦機関向けバージョンの配布を準備中**であることが報じられました。既報のNSA個別利用を超え、体系的な多機関配布を計画するものです。

## エコシステム＆プラグイン

### Clarc — 非開発者でもClaude Codeを使えるmacOSアプリ

開発者が非技術系の同僚のために作ったmacOSネイティブアプリです。Claude Code CLIをベースに、ターミナルやGitHub設定という参入障壁を取り除きました。約10MBの軽量アプリで、プロジェクト別ワークスペース、GitHub OAuth自動設定、内蔵ファイルエクスプローラーを提供します。

企画者、デザイナー、PMなど非技術者がClaude Codeのコード分析・生成能力を活用したいがターミナルが壁だった場合に有用です。

[GitHub](https://github.com/ttnear/Clarc)
