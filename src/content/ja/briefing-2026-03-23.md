### Claude Cowork Desktop — Projects機能がリリース（3/20）

Claude Cowork Desktopに**Projects**機能が追加され、ファイル・指示事項・タスクコンテキストを1つのワークスペースで管理できるようになりました。

**主な特徴：**
- **ローカルフォルダ連携**: 既存フォルダを接続するか新規作成し、ファイル・指示・作業をプロジェクトに紐付けて管理
- **セッション間コンテキスト維持**: 毎セッションごとにフォルダアクセスや目標を再設定する必要なく作業を再開可能
- **スケジュールタスク統合**: 2/25導入のScheduled TasksがProjects範囲内で直接管理可能に
- **今後の計画**: クロスデバイス同期および追加の安全性改善を予定

[CyberSecurityNews](https://cybersecuritynews.com/projects-feature-claude-cowork-desktop/) | [Times of AI](https://www.timesofai.com/news/anthropic-is-adding-projects-and-folders-to-claude-cowork/)

## 開発者ワークフローティップス

### Claude Code Channels — 実践的な自動化パターン

Channelsのリリースから1週間が経ち、コミュニティで実践的な活用パターンが確立されつつあります。

**デプロイ自動通知パターン：**
```bash
# CI/CDパイプラインでデプロイ完了後にClaudeに通知
claude --channels &
# Telegram/Discordで「デプロイ完了、スモークテスト実行して」とメッセージ送信
```

**モバイルコードレビューパターン：**
```bash
# --channels + --bareの組み合わせでリモートレビュー
ANTHROPIC_API_KEY=sk-... claude --channels --bare
# 移動中にTelegramからPRレビューを依頼 → 結果を受信
```

v2.1.81の権限リレーによりモバイルからのツール承認も完全サポートされ、完全なリモートワークフローが実現します。
