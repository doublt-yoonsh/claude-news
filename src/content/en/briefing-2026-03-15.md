## Developer Workflow Tips

### v2.1.76 MCP Elicitation — Practical Use Cases

Yesterday's MCP Elicitation feature lets MCP servers request structured input from users mid-task. Key scenarios:

- **Deployment pipelines**: MCP server confirms target environment, version, rollback conditions via form
- **Database migrations**: Request user confirmation on impact scope before schema changes
- **OAuth auth flows**: Browser URL-based authentication dialogs

Use `Elicitation`/`ElicitationResult` hooks to intercept or automate responses — enabling non-interactive workflows in CI/CD environments.

**Scale:**
- **24,000 fraudulent accounts**, over **16 million Claude queries** total

Primary targets: Claude's agentic coding, reasoning, and tool-use capabilities. A reminder of the importance of API key management and anomalous traffic monitoring.
