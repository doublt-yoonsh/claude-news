## Developer Workflow Tips

### Claude Code Channels — Control Claude Code from Telegram/Discord

Claude Code Channels launched as a research preview. Send messages directly into a running Claude Code session from Telegram or Discord.

**How it works:**
- Plugin-based — forwards messages from Telegram/Discord to local Claude Code session
- Session processes requests with full filesystem, MCP, and git access, then replies through the same app
- Slack, WhatsApp, and iMessage support requested by the community

```bash
# From Telegram to Claude Code:
# "Refactor the auth logic in src/api/users.ts"
# → Local Claude Code executes and replies back to Telegram
```

[VentureBeat](https://venturebeat.com/orchestration/anthropic-just-shipped-an-openclaw-killer-called-claude-code-channels) | [Claude Code Docs](https://code.claude.com/docs/en/channels)

### Community Best Practice — "Think in Agents, Not Prompts"

Emerging patterns from Hacker News and r/ClaudeCode:

- **Composable Skills**: Focused `SKILL.md` files paired with subagents is the new best practice
- **Spec-first approach**: One developer spent 2 hours writing a 12-step implementation doc → Claude executed step by step successfully
- **r/ClaudeCode**: 4,200+ weekly contributors, more than triple competitor communities
