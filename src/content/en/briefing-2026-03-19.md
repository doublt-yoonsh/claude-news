## New Features & Practical Usage

### Claude Code v2.1.79 — Remote Control and Stability Improvements (3/18)

Released the same day as v2.1.78, **v2.1.79** focuses on developer experience and stability.

**Key changes:**
- **`--console` auth flag** — `claude auth login --console` authenticates directly with the Anthropic Console (API billing). Streamlines auth workflows in team/enterprise environments
- **Turn duration toggle** — "Show turn duration" added to `/config` menu. See exactly how long each turn takes in real time
- **~18MB startup memory savings** — Memory efficiency improved across all scenarios
- **Non-streaming API fallback improvement** — 2-minute per-attempt timeout prevents sessions from hanging indefinitely
- **`CLAUDE_CODE_PLUGIN_SEED_DIR` multi-path support** — Specify multiple seed directories separated by `:` (Unix) or `;` (Windows)
- **`claude -p` subprocess hang fix** — No longer hangs when spawned without explicit stdin (e.g., Python `subprocess.run`)
- **Ctrl+C in `-p` mode fix** — Ctrl+C now works correctly in print mode
- **`/btw` streaming fix** — Side questions during streaming now return their own answers instead of the main agent's output
- **`SessionEnd` hooks fix** — `SessionEnd` hooks now fire correctly when switching sessions via interactive `/resume`
- **Voice mode startup fix** — Voice mode activates correctly on startup when `voiceEnabled: true` is set

**VSCode-specific:**
- **`/remote-control`** — Bridge your session to claude.ai/code to continue from a browser or phone
- **AI-generated session tab titles** — Session tabs get auto-generated titles based on your first message
- **Thinking pill fix** — Shows "Thought for Ns" instead of "Thinking" after response completes
- **Sidebar session diff button fix**

[GitHub Releases](https://github.com/anthropics/claude-code/releases)

### DOJ Labels Anthropic an "Unacceptable Risk to National Security" (3/18)

The federal government filed its **first substantive rebuttal** in the Pentagon-Anthropic lawsuit.

**Key highlights:**
- **DOJ 40-page filing** — Urges court to reject Anthropic's preliminary injunction request. Argues Anthropic might "attempt to disable its technology or preemptively alter the behavior of its model" during "warfighting operations" if it feels its corporate "red lines" are being crossed
- **First Amendment argument rejected** — DOJ argues Anthropic's refusal to accept government contract terms is conduct, not speech, and unlikely to succeed
- **Core dispute: autonomous weapons and mass surveillance** — Pentagon wants Claude for "all lawful" uses; Anthropic refuses to drop its two red lines on autonomous weapons and domestic mass surveillance
- **150 retired federal and state judges** filed amicus briefs supporting Anthropic
- **3/24 hearing D-5**: Preliminary injunction hearing before U.S. District Judge Rita Lin is 5 days away

## Developer Workflow Tips

### v2.1.79 `/remote-control` — Continue Your Session from a Browser

Bridge your VSCode Claude Code session to claude.ai/code so you can continue working from a phone or any browser when away from your desk.

```
# In the VSCode Claude Code panel
/remote-control
```

### `--console` Flag for Streamlined API Auth

Authenticate directly with Anthropic Console (API billing) from the CLI in team environments:

```bash
claude auth login --console
```

### Monitoring Turn Duration

Enable `/config` → "Show turn duration" → See how long each turn takes in real time to optimize your prompts.
