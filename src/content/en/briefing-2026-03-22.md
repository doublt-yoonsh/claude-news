## New Features & Practical Usage

### DOJ Directly Counters Anthropic's First Amendment Argument — 40-Page Filing (3/17, Key This Week)

The Department of Justice filed a **40-page brief** urging the California federal court to reject Anthropic's preliminary injunction request. Legal battle reaches peak intensity ahead of the 3/24 hearing.

**DOJ's core arguments:**
- **"Conduct, not speech"**: Anthropic's refusal to accept government contract terms is conduct, not expression protected by the First Amendment
- **"Same result even assuming retaliation"**: Even if a retaliatory motive existed, the government would have taken the same actions based solely on legitimate national security concerns
- **"Terms of service became unacceptable"**: Anthropic's terms became unacceptable to the executive branch after the company insisted on restrictions regarding autonomous weapons and mass surveillance

## Developer Workflow Tips

### Claude Code Channels — Control from Telegram in 5 Minutes

Channels, introduced as a research preview in v2.1.80, let you send messages from Telegram or Discord directly into a running Claude Code session.

```bash
# Start Claude Code with channel support
claude --channels
```

Built on MCP, so the community can build additional connectors for Slack, WhatsApp, etc. v2.1.81's `--channels` permission relay enables mobile tool approval.

### `--bare` Mode — The Key to CI/CD Automation

```bash
ANTHROPIC_API_KEY=sk-... claude -p "Review this PR" --bare
```

Skips hooks, LSP, plugin sync, and skill directory walks for fast, clean execution in automated environments.
