# Claude Code 데일리 브리핑 - 2026-02-25

## 사용 팁

- **v2.1.51: `claude remote-control` 서브커맨드 추가** (2/24)
  로컬에서 실행 중인 Claude Code 세션을 모바일, 태블릿, 브라우저(claude.ai/code)에서 이어서 제어할 수 있는 기능이 추가되었습니다. 외출 중에도 작업 흐름을 끊지 않고 이어갈 수 있습니다. 다만 현재 Pro/Max 개인 계정에서 "Contact your administrator" 오류가 표시되는 버그가 보고되어 있습니다.
  [GitHub Releases](https://github.com/anthropics/claude-code/releases) | [Claude Code Docs](https://code.claude.com/docs/en/remote-control)

- **v2.1.51: 플러그인 커스텀 npm 레지스트리 및 버전 고정 지원** (2/24)
  사설 npm 레지스트리에서 플러그인을 설치하고 특정 버전으로 고정할 수 있게 되었습니다. 기업 환경에서 승인된 플러그인만 배포하거나 버전을 통제할 때 유용합니다. 플러그인 마켓플레이스 git 타임아웃도 30초→120초로 확대되었습니다.
  [GitHub Releases](https://github.com/anthropics/claude-code/releases)

- **v2.1.53: 대규모 안정성 개선** (2/25)
  Windows 환경 크래시 다수 수정(ARM64, x64, WASM 인터프리터), `--worktree` 첫 실행 시 무시되던 버그 수정, UI 입력 후 깜빡임 현상 해결 등이 포함되었습니다. Ctrl+F 백그라운드 에이전트 일괄 종료 시 알림이 에이전트당 하나씩이 아닌 하나로 통합되었습니다.
  [GitHub Releases](https://github.com/anthropics/claude-code/releases)

## 보안/제한 이슈

- **v2.1.51: Hook 보안 취약점 2건 수정** (2/24)
  `statusLine`과 `fileSuggestion` 훅 명령의 보안 취약점이 수정되었습니다. 또한 HTTP 훅이 임의 환경변수를 인터폴레이션할 수 있던 취약점이 패치되었으며, 샌드박스 활성 시 HTTP 훅 트래픽이 샌드박스 네트워크 프록시를 경유하도록 변경되었습니다. 훅을 사용하는 환경이라면 즉시 업데이트를 권장합니다.
  [GitHub Releases](https://github.com/anthropics/claude-code/releases)
