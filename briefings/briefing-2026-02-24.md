# Claude Code 데일리 브리핑 - 2026-02-24

## 사용 팁

- **커스터마이징 설정 활용하기**: Claude Code는 37개 설정 항목과 84개 환경 변수를 지원한다. 터미널 테마, Vim 모드, 알림, 에포트 레벨(Low/Medium/High) 조정이 가능하고, `settings.json`을 Git에 체크인하면 팀 전체에 설정을 공유할 수 있다. 커스텀 에이전트는 `.claude/agents`에 마크다운으로 생성하면 된다. [출처 - GeekNews](https://news.hada.io/topic?id=26649)

- **Plan 모드로 안전하게 작업하기**: `/plan` 명령으로 Claude가 먼저 다단계 전략을 작성하고 사용자 승인을 기다리게 할 수 있다. 복잡한 리팩터링이나 대규모 변경 시 코드를 건드리기 전에 계획을 검토할 수 있어 실수를 줄인다. [출처 - Geeky Gadgets](https://www.geeky-gadgets.com/claude-code-tips-2/)

- **MCP Tool Search로 컨텍스트 절약**: MCP 서버의 Lazy Loading 기능으로 컨텍스트 사용량을 최대 95% 절감할 수 있다. 다수의 MCP 서버를 동시에 실행해도 컨텍스트 한도 걱정 없이 사용 가능하다. [출처 - Claude Code Docs](https://code.claude.com/docs/en/plugins)
